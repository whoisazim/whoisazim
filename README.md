👋 About Me

I'm Md Azim, a Full Stack MERN & Next.js Developer from Bangladesh. I build fast, animated, and scalable web apps — from recipe-sharing platforms with Stripe payments to pet-adoption systems with role-based auth. I care a lot about clean UI, smooth motion (Framer Motion), and shipping to production on Vercel. Currently leveling up into a professional Software Engineer, one real project at a time.
